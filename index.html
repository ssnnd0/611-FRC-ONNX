Raspberry Pi Machine Learning Vision for FRC

Inspiration

Vision tracking for FRC (FIRST Robotics Competition).

What does it do?

CustomVision.AI exports to Tensorflow and Onnx, which is used within my program to detect the object itself, then the Python program runs vision math on the supplied co-ordinates of any detected object to find out where it is in space. The program detects the object, and runs vision calculations to find distance to the object. It sends this information over the robot network using PyNetworkTables for additional processing on the roboRIO (e.g; pathfinding).
